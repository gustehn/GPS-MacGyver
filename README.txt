GPS - Macgyver

Two different modes:
*Online map with shortest path over GBG, showing current position.
*Offline map, rendered by program, shortest path over GBG

Start the online map through Macgyver.java
Start the offline map through Offlinemap.java
Will be optional in later version.

In this demoversion - no need for GPS.
Current pos is automatically set in to IT-University.
